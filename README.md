### 🛠 Research Focus
- Laser powder bed fusion of stainless steels and engineering alloys  
- Super duplex stainless steels, including UNS S32750 / SDSS 2507  
- Microstructure, phase balance, texture, and residual stress  
- Structure–property–corrosion relationships  
- Electrochemical characterization: OCP, potentiodynamic polarization, and EIS  
- Heat treatment, post-processing, and secondary-phase formation  
- Scientific computing and data analysis using Python and MATLAB
